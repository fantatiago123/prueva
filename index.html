¿Qué significa HTML y por qué se dice que es un lenguaje de marcas? Enumera 8 etiquetas HTML

El HTLM sirve para estructurar y desplegar en una página web sus contenidos. Y es un lenguaje de mercado ya que codifica documentos y textos e incorpora etiquetas y marcas

etiquetas HTML:

Head: Provee información general.

Title: Indica el título del documento.

Img: Representa una imagen en el documento.

Nav: Representa una sección de una página cuyo propósito es proporcionar enlaces de navegación.

Body : Representa el contenido de un documento HTLM.

Aside: Representa una sección de una página que consiste de contenido en esta directamente direccionado con el contenido principal del documento.

Audio: Se usa para insertar audio en el documento HTLM.

Footer: Representa un pie de página para el contenido de sección más cercano.

La anatomia de el HTML es: etiketa de apertura:Consiste en el nombre del elemento(en este caso, p), encerrado por paréntesis angulares (< >) de apertura y cierre.
